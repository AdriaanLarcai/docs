# Introduction

UiPath aggregates custom build _**AIForged**_\*\* \*\* activities that are available in the “Manage Packages” tab in _**UiPath Studio**_.

### Minimum Requirements

To successfully us the _AIForged_ activities in _UiPath Studio,_ the minimum license requirements of your studio should be version 2021.4.4.

### Package Name

The name of the package is: \_**AIForged** \_ by **AIForged**
